# Project Genesis

Project Genesis is the next-generation HVAC communication and monitoring platform.

## Components

### Firmware
- UnitLink-Indoor (Apollo)
- UnitLink-Outdoor (Atlas)

### Shared Libraries
- UnitLinkWireless
- UnitLinkMQTT
- UnitLinkPacket

### Server
- MQTT Broker
- Python Worker
- API Services

### Hardware
- Apollo PCB
- Atlas PCB
- Schematics
- Manufacturing Files

### Documentation
- Protocol Specifications
- Hardware Documentation
- API Documentation
- Development Milestones

## Development Philosophy

- Modular
- Shared codebase
- Private Git repository
- Version controlled from day one
- Build first, optimize second
